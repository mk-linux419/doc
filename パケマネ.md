# パケマネ
## The zen of fpkg
- ユーザーフレンドリー
- シンプル
- パッケージの作成がしやすい

## 名前
fpkg ~~とかでいいやろ~~ (未定)

## 使い方
```
fpkg [オプション] [コマンド]
```

## コマンド
### パッケージ追加系
- install
```
fpkg install [パッケージ名/パッケージファイルのパス]
```
or
```
fpkg i [パッケージ名/パッケージファイルのパス]
```

- sync
```
fpkg sync
```
or
```
fpkg s
```

- upgrade
```
fpkg upgrade
```
or
```
fpkg u
```

#### 高度な使い方
pacman -Syu 的なやつ
```
fpkg sync+upgrade
```
or
```
fpkg su
```
